# Project roadmap — Smart Order UI

One milestone at a time. Do not skip ahead.
Update the Current Milestone in CLAUDE.md every time you finish one.

---

## Milestone 1 — Hello World ← YOU ARE HERE
**What you build:** A single page with text and a button. Nothing else.
**What you learn:** Vite setup, JSX, Tailwind basics, using Cursor chat.
**File:** `docs/MILESTONE_1.md`

---

## Milestone 2 — Components and Props
**What you build:** Break App.jsx into small reusable components.
A Navbar, a Hero section, a Card component.
**What you learn:** How components talk to each other via props.
Understand why React uses components instead of one big file.
**File:** `docs/MILESTONE_2.md` (created when you finish M1)

---

## Milestone 3 — State and Interaction
**What you build:** Add a button that does something — a counter,
a toggle, a simple form. Make the page interactive.
**What you learn:** useState hook, event handlers, conditional rendering.
**File:** `docs/MILESTONE_3.md` (created when you finish M2)

---

## Milestone 4 — Fetch Real Data
**What you build:** Call a free public API (e.g. a product list) and
display the results on screen. Add a loading spinner and error message.
**What you learn:** useEffect, fetch/axios, async/await in React,
handling loading and error states.
**File:** `docs/MILESTONE_4.md` (created when you finish M3)

---

## Milestone 5 — Connect to Your Spring Boot backend
**What you build:** Replace the public API with your actual backend.
Show real products from your Spring Boot service.
**What you learn:** Axios client setup, environment variables, CORS.
**File:** `docs/MILESTONE_5.md` (created when you finish M4)

---

## Milestone 6 — Place an Order
**What you build:** A form to place an order. POST to your backend.
Show a success or error message after submission.
**What you learn:** Forms in React, useMutation with React Query,
controlled inputs, form validation.
**File:** `docs/MILESTONE_6.md` (created when you finish M5)

---

## Milestone 7 — Order Status Page
**What you build:** A page to look up an order by ID and see its status
with a visual progress tracker.
**What you learn:** React Router, URL params, polling for updates.
**File:** `docs/MILESTONE_7.md` (created when you finish M6)

---

## Rule
Only create the next milestone doc when the current one is fully done.
This keeps the project focused and avoids overwhelm.
